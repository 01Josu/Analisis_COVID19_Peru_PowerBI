# Análisis COVID-19 Perú – Power BI

Proyecto de análisis de datos de COVID-19 en Perú, incluyendo métricas sobre positividad de casos, hospitalizaciones, disponibilidad de camas UCI, defunciones y vacunación.

## Tecnologías usadas
- Power BI Desktop
- Power Query para limpieza de datos
- DAX para métricas calculadas
- Extracción de datos de fuentes web oficiales

## Funcionalidades
- Dashboards interactivos con KPIs, gráficos y tooltips
- Análisis por región y periodo
- Visualización clara para la interpretación de información
